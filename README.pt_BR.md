## ✨ O Que Você Vai Construir

Digite dois nomes de usuário do GitHub e veja seus gráficos de contribuição se enfrentarem em uma arena de batalha pixel art. Construído com **Astro v5**, estilizado com a icônica fonte retrô **Press Start 2P** e alimentado pelos dados de contribuição do GitHub.

> 🏆 Este é o **ponto de partida** — você vai construir tudo passo a passo, guiado pelo GitHub Copilot.

## 📚 Trilhas do Workshop

Escolha o caminho que melhor se adapta ao seu fluxo de trabalho:

| Trilha | Ferramentas | Ideal Para |
|--------|------------|-----------|
| 🖥️ **VS Code** | Chat, Plan Mode, Agent Mode, background agents, revisão nativa no editor | Desenvolvedores orientados ao editor |
| 💻 **CLI** | `copilot`, `@file`, `/plan`, `/fleet`, `/delegate`, `/review` | Usuários avançados de terminal |

## 🚀 Início Rápido

1. **Crie seu próprio repositório** — clique em **Use this template** ou faça um fork deste repositório.
2. **Escolha sua trilha:**
   - **VS Code:** clone seu repositório e abra no VS Code.
   - **GitHub Copilot CLI:** clone seu repositório, instale o `copilot` e abra seu terminal.
3. **Inicie o workshop** → [workshop/pt_BR/00-overview.md](workshop/pt_BR/00-overview.md)
