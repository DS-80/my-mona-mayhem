## ✨ What You'll Build

Enter two GitHub usernames and watch their contribution graphs go head-to-head in a pixel-art battle arena. Built with **Astro v5**, styled with the iconic **Press Start 2P** retro font, and powered by GitHub's contribution data.

> 🏆 This is the **starting point** — you'll build the whole thing step by step, guided by GitHub Copilot.

## 🚀 Quick Start

1. **Create your own repo** — click **Use this template** or fork this repository.
2. **Pick your track:**
   - **VS Code:** clone your repo and open it in VS Code.
   - **GitHub Copilot CLI:** clone your repo, install `copilot`, and fire up your terminal.
3. **Start the workshop** → [workshop/00-overview.md](workshop/00-overview.md)
